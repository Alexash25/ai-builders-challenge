Initial Plan and Idea - AI Creative Director for Video
Creator can upload:
    - A short video
    - Optional script/outline
    - Target audience
    - Target platform
    - Desired tone and video length
    - Other natural language parameters

This platform will analyze video speech, visuals, audio quality, pacing, structure, and alignment with the creator's goal
Will return
    - Timestamped feedback
    - Prioritized editing plan
    - Suggested script changes
    - Optional automatic corrections
    - Generated preview of improved edit

What the AI should evaluate
    1. Story and message
    2. Speech and audio
    3. Visual presentation
    4. Multimodal alignment
    5. Accessibility
